Telegram WebOS Shortcut for LG TV
Telegram WebOS App

A lightweight shortcut to open Telegram on your LG WebOS TV.

How to Use

Download the prebuilt .ipk file from this repository.

Enable Developer Mode on your TV and connect it via webOS CLI.

Install the app using:

ares-install --device tv2 ~/webapp-telegram/webapp/com.telegram.web_1.0.0_all.ipk

The app opens Telegram in full screen on your TV.
